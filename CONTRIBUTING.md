# Contributing

Contributions are welcome. Keep the converter independent from proprietary Ostrid editor code.

1. Fork the repository and branch from `main`.
2. Run `bun install`.
3. Add focused tests for inference or schema changes.
4. Run `bun run check`, `bun run check-types`, `bun test`, and `bun run build`.
5. Open a pull request explaining the CAD convention and sample used.

Only contribute code you are entitled to license under GPL-3.0-or-later. Do not commit customer drawings, private plans, access tokens, or proprietary editor source.
